# DSA learning

✅ Simple way to think about it

|Level|	Meaning|
|-----|--------|
|Data Structure|	Where data is stored|
|Pattern|	How you approach the problem|
|Algorithm|	Exact steps used to solve it|

**A simple summary:**
- **Pattern (Strategy) :** 
  - Pattern to decide which Algorithm or data structure logic to apply.
  - I should probably use a 'Sliding Window' approach here because the problem mentions a sub-array.
- **Algorithm (Recipe) :** Okay, now I will write the for loop and the if statements to actually move that window.
